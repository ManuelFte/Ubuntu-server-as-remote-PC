# VPS as remote PC

## Description

Turns a VPS into a basic remote PC. Installed software:

* LXDE
* Chromium
* OpenDNS
* DWService (downloads script, doesn't install)

## Usage

```
git clone https://ManuelFte@bitbucket.org/ManuelFte/vps-as-remote-pc.git
cd vps-as-remote-pc
sh vps-as-remote-pc.sh
```
